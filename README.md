# TIPiS

# Описание
Система предсказывает вероятность победы домашней команды в матче NBA на основе исторической статистики.
Пользователь выбирает две команды через интерфейс — модель возвращает вероятности и вердикт.

Разработано как учебный проект по машинному обучению и анализу спортивных данных.

# Модель
Алгоритм: XGBoost (градиентный бустинг деревьев).
Используемые метрики
Для каждой команды берутся средние значения:

FG% — попадания с игры;
FT% — штрафные броски;
3PT% — трёхочковые;
AST — передачи;
REB — подборы;
Всего 10 признаков на матч.

# Гиперпараметры:
n_estimators=200;
max_depth=5;
learning_rate=0.05;
Оценка качества: accuracy, ROC-AUC;
# Прогноз
> 55% → домашняя команда выиграет
< 45% → гостевая команда выиграет
45–55% → матч равный (учитывает домашний фактор и неопределённость)
